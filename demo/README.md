# Demo

A Next.js app to test the `@caprionlinesrl/puck-plugin-media` plugin during development.

## Run

From the repository root:

```bash
yarn workspace demo dev
```

Open [http://localhost:3000](http://localhost:3000)

## Notes

- Uses library source files directly (no build required)
- Editor state persists to localStorage
- Mock API simulates image, gallery, and document endpoints
